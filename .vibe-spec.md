# Vibe Spec: The Bunchy Tops

## Mood
**Adjectives**: warm, hazy, unhurried  |  **Anti-mood**: hyped, clinical, tropical-resort
**Archetype**: "the porch at dusk" — a gathering place where the world slows down

## Color
- Primary: #1C3738 (Deep Swell — deep ocean foundation, anchors every dark surface)
- Secondary: #C16E4F (Sunbaked Clay — terracotta warmth, used for CTAs and accent blocks)
- Accent: #DCA74E (Golden Hour — muted gold, hover states, highlighted text, decorative)
- Accent 2: #7B9095 (Wistful Blue — hazy sky, subheadings, secondary UI elements)
- Surface Light: #EAE3D8 (Canvas/Sand — parchment warmth, about section bg)
- Surface Dark: #1C3738 (Deep Swell — music section bg)
- Neutral Mid: #8A8175 (Driftwood — muted warm grey, secondary text, dividers)
- Text: #2B2B2B (Charcoal — NEVER pure black)
- Palette strategy: analogous warm (clay + gold) against a cool anchor (deep swell), split by warm neutral

## Typography
- Display: DM Serif Display (Google Fonts substitute for Recoleta), lowercase only, weight 400, tracking -0.02em to -0.03em
- Body: Space Grotesk, weights 300/400/500, leading 1.6
- Scale ratio: 1.333 (perfect fourth) → 12px / 16px / 21px / 28px / 37px / 50px / 67px
- Body min: 16px. Hero headline: clamp(48px, 8vw, 96px)

## Spacing
- Base unit: 8px  |  Density: spacious (breathing room is the brand)
- Section padding: 96px desktop / 56px mobile
- Component gap: 32–48px

## Motion
- Micro: 200ms ease-out  |  Entrance: 600ms cubic-bezier(0.4, 0, 0.2, 1) + translateY(24px) → 0
- Stagger: 100ms between grouped items
- Parallax: hero image at 0.4 parallax ratio
- Respect prefers-reduced-motion

## Layout
- Grid: 12-col fluid, max-width 1280px, gutters 24px
- Hero: full-viewport image-bg, centered text overlay
- About: 2-col (text left, image right) on desktop, stacked on mobile
- Music: 2-col card grid on desktop, stacked on mobile
- Gallery: CSS columns masonry, 3-col desktop, 2-col tablet, 1-col mobile
- Breakpoints: 320 / 640 / 768 / 1024 / 1280 / 1440

## Film Aesthetic
- Film grain: SVG noise filter pseudo-element, opacity 0.035–0.06
- Image color treatment: CSS filter sepia(0.25) hue-rotate(-10deg) saturate(0.9) brightness(0.95) contrast(0.9)
- Lifted blacks: CSS levels via brightness + contrast to prevent true black
- All images share the same CSS filter class for instant brand cohesion
